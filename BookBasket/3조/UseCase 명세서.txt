1.	로그인/아웃
		체인점코드, 아이디, 비번을 이용해서 로그인/아웃 처리

2.	고객등록기능
		첫 번째 내방 고객에 대하여 서류로 정보를 받아서 나중에 입력하는 방식으로 처리

3.	고객정보확인
		이름검색, 핸드폰검색을 기준으로 검색 실시
		-->	마지막 시술내역 자동 조회기능 추가

4.	고객시술내역 등록 기능
		고객정보확인 기능을 이용하여 고객을 검색한 후
		시술내역을 정형화하여 데이터베이스에 저장하도록 한다.

5.	고객시술내역 확인
		핸드폰 검색을 기준으로 하여 고객 시술내역 확인
		전체검색 검색을 하도록 한다.
=============================================================================
1.	디자이너 등록 기능
		서류상으로 디자이너 정보를 제공받아서 일률적으로 등록하도록 한다.

2.	디자이너 정보 변경/삭제 기능
		서류상으로 디자이너 정보를 제공받아서 일률적으로 등록하도록 한다.

3.	디자이너 정보 확인 기능
		이름, 전번을 이용한 검색 기능

4.	디자이너 근태 등록
		출근, 퇴근 만 체크하고
		10시 이전 출근은 출근으로 처리하고 10시 이후 출근은 지각으로 처리한다.
		9시 이전 퇴근은 조퇴로 처리하고 9시 이후 퇴근은 퇴근으로 처리한다.

5.	디자이너 근태 확인
		디자이너별, 월단위 검색 처리한다.

==========기능 이동
1.	프로그램을 시작하면 로그인 화면이 나온다.
2.	로그인에 성공하면 메인화면이 나오면서 기본적으로 고객관리 메인 화면이 출력된다.
3,	고객관리 메인화면은 기본적으로 고객정보 화면이 나온다.

고객 정보 화면

4.	고객 등록 단추를 누르면 고객 등록 대화상자가 출력된다.
5.	고객 등록이 성공하면 대화상자가 자동으로 사라진다.
6.	검색 단추를 누르면 해당 검색에 조건에 맞는 고객 정보가 테이블에 출력된다.
7.	출력결과중 하나를 선택하면 그 정보가 텍스트 필드에 출력된다.
8.	시술등록 단추를 누르면 테이블에서 선택한 고객을 대상으로 하는 시술등록 대화상자가 출력된다.
9.	시술등록이 성공하면 대화상자가 없어진다.
10.	정보변경 단추를 누르면 입력상자에 수정한 내용으로 정보가 변경된다.

고객시술확인 화면
1.	검색 단추를 누르면 검색조건에 따라 검색 결과가 테이블에 출력된다.
2.	테이블에서 특정 내용을 출력하면 시술 내역의 코맨드 부분이 텍스트 에리어에 출력된다.


======================================
디자이너 관리
1.	디자이너 관리 메인은 디자이너 정보 확인 화면을 기본으로 한다.
2.	등록단추를 누르면 디자이너 등록 대화상자가 나와서 디자이너 등록을 하도록 한다.
3.	등록이 완료되면 대화상자가 사라진다.
4.	검색 단추를 누르면 검색 조건에 맞는 디자이너 정보가 테이블이 출력된다.
5.	테이블의 디자이너를 선택하면 정보가 텍스트 필드에 나오게 한다.
6.	정보변경 단추를 누르면 정보가 변경되도록 한다.
7.	삭제 단추를 누르면 디자이너가 삭제되도록 한다.
8.	테이블에서 디자이너를 선택하고
	출근, 퇴근 단추를 누르면 출퇴근 정보가 기억된다.
9.	디자이너 근테 확인 화면에서
	검색 단추를 누르면 검색 결과가 테이블에 출력되도록 한다.
